QDirWeiGraph

Plot Directed Weighted Graphs

QDirWeiGraph is a Qt C++ based program able to draw your directed weighted graphs from XML files.

Available actions :

  • Create/Save/Open graphs with XML format
  • Move nodes. Choose nodes labels.
  • Use different displays. Solid, dashed or dotted arrows
  • Export as .svg
